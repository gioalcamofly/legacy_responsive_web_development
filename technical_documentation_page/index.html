Variables

A Python variable is a symbolic name that is a reference or pointer to an object. Once an object is assigned to a variable, you can refer to the object by that name. But the data itself is still contained within the object.

a = 1 b = "hi" c = [1, 2, 3]

In Python, you do not need to declare the type of the object to store in the variable.

You can also cast variables to other types. For example:

a = "3" b = int(a) print(type(b)) # returns int
Control Structures

The main control structures in python are if and for.

if lets you control if some staments are executed or not:

if a: print('hello') else: print('hola')

In this case, if a is evaluated to True then it will print "Hello". But if is turns out to be False it will print "Hola".
